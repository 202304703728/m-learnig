1) Instalar o composer

    1.1) instalador do Windows

    O instalador - que requer o PHP instalado - fará o download do Composer
     e configurará sua variável de ambiente PATH para que seja possível chamar o Composer
     de qualquer diretório.

    Baixer e executar Composer-Setup.exe - ele instalará a versão mais recente do composer sempre que
    for executado.

    1.2) Instalação de linha de comando

    Para instalar rapidamente o Composer no diretório atual, executar o seguinte script no terminal:

    php -r "copy('https://getcomposer.org/installer', 'composer-setup.php');"
    php -r "if (hash_file('sha384', 'composer-setup.php') === 'e21205b207c3ff031906575712edab6f13eb0b361f2085f1f1237b7126d785e826a450292b6cfd1d64d92e6563bbde02') { echo 'Installer verified'; } else { echo 'Installer corrupt'; unlink('composer-setup.php'); } echo PHP_EOL;"
    php composer-setup.php
    php -r "unlink('composer-setup.php');"

2) Para manipular arquivos PDF, baixar a biblioteca PDFParser

composer require smalot/pdfparser


3) Iniciar o servidor integrado - development (prompt do terminal):

php -S localhost:8000
